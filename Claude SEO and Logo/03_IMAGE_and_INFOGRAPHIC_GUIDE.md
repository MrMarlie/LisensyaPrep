## SHOULD YOU ADD IMAGES? YES — HERE'S WHY

Images serve three purposes for LisensyaPrep:
1. SEO: Google image search can send you additional traffic
2. AdSense: Pages with images have lower bounce rates = more ad impressions
3. E-E-A-T: Original graphics signal that real humans created the content

## THE RULE: Never use stock photos of "generic nurses" or "generic students."
   They look fake. Use infographics and data visuals instead.
   These are UNIQUE to your site and impossible for competitors to copy.

### TYPE 2: FEATURED ARTICLE IMAGES
These appear at the top of each article. Use Canva templates.
Do NOT use generic stock photos. Instead use:
  - Text-based thumbnail: Dark background + white text title + profession icon
  - Example: Dark blue background, stethoscope icon, "NLE 2026 — How to Pass on First Take"
  - This makes your articles look branded and professional

Recommended Canva template search: "Blog thumbnail education dark"

File naming convention:
  /images/articles/nle-[article-topic].jpg
  /images/articles/cle-[article-topic].jpg
  /images/articles/let-[article-topic].jpg
  /images/articles/pharmacy-[article-topic].jpg
  /images/articles/medtech-[article-topic].jpg
  /images/articles/agriculture-[article-topic].jpg

### TYPE 3: AUTHOR PHOTOS
Real author photos dramatically improve E-E-A-T scores.
  - Use a clean headshot (not a graduation photo)
  - Display name + credentials (RN, RCrim, RPh, etc.)
  - Store at: /images/authors/[name].jpg
  - Size: 200x200px, circular crop in CSS

## IMPORTANT IMAGE SEO RULES

1. Every image MUST have descriptive alt text (see examples above)
2. File names must be descriptive: nle-12-week-study-plan.png NOT image001.png
3. Compress all images before uploading — use TinyPNG.com (free)
4. Target file size: Under 150KB per image
5. Use WebP format if your site supports it (faster loading = better SEO)
6. Add your logo/watermark to every infographic — prevents hotlinking and builds brand
